pour installer pawns suvez les commandes suivantes :

Sudo apt install docker


docker pull iproyal/pawns-cli:latest
docker run --restart=on-failure:5 iproyal/pawns-cli:latest -email="email@example.com" -password="change_me" -device-name=raspberrypi -device-id=raspberrypi1 -accept-tos


Remplacez email=email@example.com par votre adresse email.
Remplacez password=change_me par votre mot de passe .
